Personal Information - Kişisel bilgi

• The aim of this project is to collect typing patterns (especially timing) of users over several sessions. This data will be used for research purposes and will only be shared with other researchers.

• Nonetheless, please choose a password that you do NOT normally use in other applications or one that is special for you, for security reasons. The collected data will be kept strictly confidential.


• Bu projenin amacı, çoklu oturumlarda katılımcıların yazım tarzınızı (özellikle zamanlama) toplamaktır. Bu veriler araştırma amaçlı kullanılacak ve sadece araştırmacılarla kullanılacaktır.

• Yine de, lütfen başka uygulamalarda kullandığınız, size özel şifreleri KULLANMAYIN. Toplanan veriler tamamiyle gizli tutulacaktır.

• Projenin bu bölümü, sayısal şifreleri girerken yazım kalıpları hakkında veri toplamayı amaçlamaktadır. Bu amaçla sizden SADECE 0-9 rakamları içeren 6 haneli bir şifre oluşturmanızı rica ediyoruz.

Remember: The password should ONLY contain numbers from 0-9.

Unutmayın: şifrenizi SADECE 0-9 arası sayılar içermelidir.
