Keystroke biometrics

Keystroke biometrics can be used to increase the security of online applications, by verifying the users' typing dynamics while entering passwords and other required inputs. For example, the system can check the typing speed of users' 4 or 6 digit numeric password in online banking.

Abstract


The objective of this project is to develop software that aims to upgrade user verification using keystroke biometrics, with a focus on Turkish keyboard.

• The software will have the potential to be used by different online platforms, such as banks, to improve their security.

• This study will collect a dataset from Turkish keyboard users, extract certain features from the raw data, and evaluate it.
